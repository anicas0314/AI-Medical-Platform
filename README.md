## üöÄ Getting Started

### 1Ô∏è‚É£ Clone the Repository

```bash
https://github.com/QwertyFusion/SwiftCare.ai.git
cd SwiftCare.ai
```

### 2Ô∏è‚É£ Set Up Environment Variables in the `root` folder

Create a `.env` file and fill in the required fields as per `.env.example`.

### 3Ô∏è‚É£ Install Dependencies in `root` folder for backend dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

### 4Ô∏è‚É£ Install Dependencies in `frontend` folder for frontend dependencies

```bash
cd .\frontend\
```

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

### 5Ô∏è‚É£ Run the backend server from the `root` folder

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Your backend should be running at [http://localhost:3000](http://localhost:3000)!

### 6Ô∏è‚É£ Run the frontend server from the `frontend` folder

```bash
cd .\frontend\
```

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Now, open [http://localhost:5173](http://localhost:5173) in your browser to see **SwiftCare.ai** in action!
